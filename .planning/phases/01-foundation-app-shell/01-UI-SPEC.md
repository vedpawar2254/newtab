# Phase 1 — UI Design Contract

> Visual and interaction contract for the Foundation + App Shell phase. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none |
| Preset | not applicable |
| Component library | radix-ui (primitives only: dialog, tooltip) |
| Icon library | lucide-react |
| Font | JetBrains Mono (with `ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, monospace` fallback) |

**Rationale:** JetBrains Mono chosen over Fira Code for wider glyph coverage and cleaner readability at small sizes. Over system monospace for visual consistency across platforms. Loaded via Google Fonts with `font-display: swap` to avoid FOIT.

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, border offsets |
| sm | 8px | Compact element spacing, icon-to-label gaps |
| md | 16px | Default element padding, sidebar item padding |
| lg | 24px | Section padding inside sidebar and content area |
| xl | 32px | Layout gaps between major regions |
| 2xl | 48px | Content area top/bottom margins |
| 3xl | 64px | Not used in Phase 1 |

Exceptions: Sidebar width is fixed at 240px (not on the spacing scale). Sidebar toggle animation uses CSS `translate` over 240px.

---

## Typography

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Body | 14px | 400 (regular) | 1.5 | Default text, sidebar items, status messages |
| Label | 12px | 400 (regular) | 1.4 | Section headers in sidebar, metadata text, toast messages (differentiated by uppercase + 0.05em letter-spacing) |
| Heading | 20px | 600 (semibold) | 1.2 | Page title placeholder in main content area |
| Display | 28px | 600 (semibold) | 1.2 | Not used in Phase 1 (reserved for future editor H1) |

**Font family:** `'JetBrains Mono', ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, 'Liberation Mono', 'Courier New', monospace`

**Notes:**
- All UI text uses this single monospace stack -- no secondary sans-serif font in Phase 1
- Letter-spacing: 0 for body, -0.01em for heading/display sizes
- Weights declared: 400 (regular) and 600 (semibold) only -- load exactly these two weights from Google Fonts

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#191919` | Page background, main content area background |
| Secondary (30%) | `#252525` | Sidebar background, skeleton placeholder base, toast background |
| Accent (10%) | `#5B9BD5` | See reserved-for list below |
| Destructive | `#E5484D` | Destructive actions only (none in Phase 1) |

### Extended Palette

| Token | Value | Usage |
|-------|-------|-------|
| surface-elevated | `#2F2F2F` | Hover state on sidebar items, active sidebar item background |
| border | `rgba(255, 255, 255, 0.08)` | All borders -- sidebar divider, content area border |
| text-primary | `#ECECEC` | Primary text on dark backgrounds |
| text-secondary | `rgba(255, 255, 255, 0.50)` | Muted text -- timestamps, status text, empty state body |
| text-accent | `#5B9BD5` | Accent-colored text (links, active navigation label) |
| skeleton-base | `#252525` | Skeleton placeholder base color |
| skeleton-highlight | `#2F2F2F` | Skeleton shimmer highlight color |
| toast-bg | `#252525` | Toast notification background |
| toast-border | `rgba(255, 255, 255, 0.12)` | Toast notification border |
| success | `#30A46C` | "Saved" status indicator text |

### Accent Reserved For

Accent color `#5B9BD5` is used exclusively on these elements:
1. Active sidebar navigation item text/indicator
2. Sidebar toggle button icon on hover
3. "New Page" action text/icon in sidebar
4. Links and interactive text references
5. Focused input ring (future phases)

Accent is NOT used for: backgrounds, large surfaces, body text, borders, or decorative elements.

### Sidebar

| Property | Value |
|----------|-------|
| Width | 240px |
| Background | `#252525` |
| Right border | 1px solid `rgba(255, 255, 255, 0.08)` |
| Default state | Open (visible) |
| Toggle animation | `transform: translateX()` over 250ms with `ease-in-out` timing |
| Content push | Main content area shifts right/left with sidebar (not overlay) |
| Item padding | 8px vertical, 16px horizontal |
| Item hover | Background `#2F2F2F`, transition 150ms |
| Active item | Text color `#5B9BD5`, background `rgba(91, 155, 213, 0.08)` |

### Sidebar Toggle Button

| Property | Value |
|----------|-------|
| Position | Fixed, top-left corner of main content area (or overlapping sidebar edge) |
| Size | 28px x 28px |
| Icon | `PanelLeftClose` (lucide) when open, `PanelLeft` (lucide) when closed |
| Accessible label | `aria-label="Close sidebar"` when open, `aria-label="Open sidebar"` when closed |
| Background | Transparent, `#2F2F2F` on hover |
| Border-radius | 4px |
| Transition | 150ms background-color |

### Main Content Area

| Property | Value |
|----------|-------|
| Background | `#191919` |
| Min-width | Fills remaining viewport width after sidebar |
| Padding | 48px top, 32px horizontal |
| Max content width | 720px centered (future editor constraint, establish now) |

## Skeleton UI Contract

Skeleton placeholders render immediately on load before data hydration.

| Property | Value |
|----------|-------|
| Animation | Shimmer (left-to-right gradient sweep) |
| Duration | 1.5s per cycle, infinite loop |
| Base color | `#252525` |
| Highlight color | `#2F2F2F` |
| Border-radius | 4px |
| Gradient angle | 90deg (horizontal sweep) |

### Skeleton Regions

| Region | Shape | Dimensions |
|--------|-------|------------|
| Sidebar title | Rectangle | 120px x 16px, 16px from top-left |
| Sidebar items (x4) | Rectangle | 180px x 12px, 8px gap between each |
| Content title | Rectangle | 280px x 24px, centered |
| Content lines (x3) | Rectangle | 400px x 14px, 8px gap between each |

## Interaction States

### Focusable Elements

| State | Style |
|-------|-------|
| Default | No outline |
| Focus-visible | 2px solid `#5B9BD5`, 2px offset, border-radius 4px |
| Hover | Per-element (see sidebar items, toggle button above) |
| Active/pressed | Opacity 0.8, 100ms transition |
| Disabled | Opacity 0.4, cursor not-allowed |

### Keyboard Navigation

| Key | Action |
|-----|--------|
| `Cmd+\` or `Ctrl+\` | Toggle sidebar open/closed |
| `Tab` | Move focus through interactive elements |
| `Escape` | Close sidebar if focused within it (future: close modals) |

## Animation Contract

All animations respect `prefers-reduced-motion: reduce` -- when enabled, all transitions resolve instantly (duration: 0ms).

| Animation | Duration | Easing | Property |
|-----------|----------|--------|----------|
| Sidebar slide | 250ms | ease-in-out | transform |
| Sidebar item hover | 150ms | ease | background-color |
| Toast enter | 200ms | ease-out | transform, opacity |
| Toast exit | 150ms | ease-in | opacity |
| Save status fade-in | 200ms | ease | opacity |
| Save status fade-out | 800ms | ease | opacity |
| Skeleton shimmer | 1500ms | linear | background-position (infinite) |
| Focus ring | 150ms | ease | box-shadow |

