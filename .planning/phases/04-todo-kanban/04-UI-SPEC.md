# Phase 4 — UI Design Contract

> Visual and interaction contract for the Todo + Kanban phase. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none |
| Preset | not applicable |
| Component library | radix-ui (primitives only: checkbox) |
| Icon library | lucide-react |
| Font | JetBrains Mono (with `ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, monospace` fallback) |

**Source:** Inherited from Phase 1 UI-SPEC (approved 2026-03-19). No changes to design system tooling.

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, checkbox-to-label gap, card inline padding |
| sm | 8px | Todo item vertical padding, card gap within column, column header padding |
| md | 16px | Panel internal padding, column padding, card padding |
| lg | 24px | Kanban board horizontal padding, panel section spacing |
| xl | 32px | Kanban board top padding |
| 2xl | 48px | Not used in Phase 4 |
| 3xl | 64px | Not used in Phase 4 |

Exceptions:
- Todo panel width is fixed at 280px (not on the spacing scale, declared in CONTEXT.md)
- Kanban column width is fixed at 280px (matches todo panel for visual consistency)
- Touch target minimum: 32px height for todo items and kanban cards

---

## Typography

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Body | 14px | 400 (regular) | 1.5 | Todo item text, kanban card title, column card count |
| Label | 12px | 400 (regular) | 1.4 | Panel header label ("TODOS"), column header label, empty state body, metadata (uppercase + 0.05em letter-spacing for headers) |
| Heading | 20px | 600 (semibold) | 1.2 | Not used in Phase 4 (no page-level headings in panel/board) |
| Display | 28px | 600 (semibold) | 1.2 | Not used in Phase 4 |

**Notes:**
- Column header names use 12px uppercase label style
- Kanban card count badge uses 12px label at text-secondary color
- Todo item text uses 14px body -- completed items get `text-decoration: line-through` and `opacity: 0.5`

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#191919` | Kanban board background (main content area) |
| Secondary (30%) | `#252525` | Todo panel background, kanban column background, kanban card background |
| Accent (10%) | `#5B9BD5` | See reserved-for list below |
| Destructive | `#E5484D` | Delete todo confirmation, delete column confirmation |

### Extended Palette (inherited from Phase 1, used in Phase 4)

| Token | Value | Usage |
|-------|-------|-------|
| surface-elevated | `#2F2F2F` | Card hover state, column header hover, add-task input focus background |
| border | `rgba(255, 255, 255, 0.08)` | Todo panel left border, kanban column borders, card borders |
| border-strong | `rgba(255, 255, 255, 0.12)` | Drag-over drop zone indicator border |
| text-primary | `#ECECEC` | Todo item text, card title, column header |
| text-secondary | `rgba(255, 255, 255, 0.50)` | Placeholder text, card count, empty state body, completed todo text |
| success | `#30A46C` | Checkbox checked state fill |

### Accent Reserved For

Accent color `#5B9BD5` is used exclusively on these Phase 4 elements:
1. "Kanban Board" sidebar navigation entry when active
2. "Add todo" action icon/text in panel header
3. "Add column" button text
4. Drag handle icon on hover
5. Focus ring on todo input and card edit input (2px solid, 2px offset)

Accent is NOT used for: kanban column backgrounds, card backgrounds, checkbox borders at rest, panel background, or decorative elements.

## Drag-and-Drop Contract

### Todo Panel Drag

| Property | Value |
|----------|-------|
| Library | dnd-kit (consistent with Phase 3 sidebar drag) |
| Trigger | Drag handle icon (appears on hover) |
| Feedback | Item lifts with slight shadow, placeholder shows gap |
| Drop indicator | 2px solid `#5B9BD5` line at drop position |
| Animation | 200ms ease for reorder settle |

### Kanban Card Drag

| Property | Value |
|----------|-------|
| Trigger | Click and drag anywhere on card |
| Feedback | Card rotates 2deg, opacity 0.6, elevated shadow |
| Column drop zone | Column background transitions to `rgba(91, 155, 213, 0.04)` when card hovers over it |
| Drop zone border | 2px dashed `rgba(255, 255, 255, 0.12)` changes to 2px solid `#5B9BD5` on hover |
| Card placeholder | Dashed border card-sized placeholder at drop position |
| Animation | 250ms ease for card settle into new position |
| Cross-column | Cards can be dragged between any columns |

## Interaction States

### Focusable Elements

| State | Style |
|-------|-------|
| Default | No outline |
| Focus-visible | 2px solid `#5B9BD5`, 2px offset, border-radius matching element |
| Hover | Per-element (see individual component tables above) |
| Active/pressed | Opacity 0.8, 100ms transition |
| Disabled | Opacity 0.4, cursor not-allowed |

### Keyboard Navigation

| Key | Context | Action |
|-----|---------|--------|
| `Enter` | Todo input focused | Add new todo |
| `Enter` | Card edit mode | Save card title, exit edit mode |
| `Escape` | Card edit mode | Cancel edit, revert to previous title |
| `Escape` | Column rename | Cancel rename, revert to previous name |
| `Delete` / `Backspace` | Todo item focused | Delete todo |
| `Space` | Todo item focused | Toggle checkbox |
| `Tab` | Todo panel | Move focus between todo items |
| `Cmd+Shift+T` / `Ctrl+Shift+T` | Global | Toggle todo panel |

## Animation Contract

All animations respect `prefers-reduced-motion: reduce` -- when enabled, all transitions resolve instantly (duration: 0ms).

| Animation | Duration | Easing | Property |
|-----------|----------|--------|----------|
| Todo panel slide | 250ms | ease-in-out | margin-right |
| Todo item hover | 150ms | ease | background-color |
| Todo complete fade-out | 300ms | ease-out | opacity (after 2s delay) |
| Checkbox check | 150ms | ease | background-color, border-color |
| Card hover | 150ms | ease | border-color |
| Card drag lift | 200ms | ease-out | transform, opacity, box-shadow |
| Card drop settle | 250ms | ease | transform |
| Column drop zone highlight | 200ms | ease | background-color, border-color |
| Add card expand | 200ms | ease-out | height, opacity |
| Kanban horizontal scroll | native | native | scroll (smooth-scroll CSS) |
