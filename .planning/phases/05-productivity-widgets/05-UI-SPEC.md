# Phase 5 — UI Design Contract

> Visual and interaction contract for the Productivity Widgets phase. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none |
| Preset | not applicable |
| Component library | radix-ui (primitives only: tooltip) |
| Icon library | lucide-react |
| Font | JetBrains Mono (with `ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, monospace` fallback) |

Source: Phase 1 UI-SPEC (approved). No changes for Phase 5.

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, habit dot gaps |
| sm | 8px | Compact element spacing, sidebar widget item vertical padding |
| md | 16px | Default element padding, widget section padding horizontal |
| lg | 24px | Section padding, gap between widget sections in sidebar |
| xl | 32px | Layout gaps between major regions |
| 2xl | 48px | Content area top/bottom margins |
| 3xl | 64px | Not used in Phase 5 |

Exceptions:
- Pomodoro circular ring: 96px diameter (sidebar-appropriate size within 240px width sidebar). Not on spacing scale but consistent with power-of-2 sizing.
- Touch target minimum: 32px for all interactive controls (start/pause/reset buttons, habit checkboxes).
- Habit dot: 8px diameter with 4px gap between dots.

---

## Typography

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Body | 14px | 400 (regular) | 1.5 | Habit names, journal entry titles, quote text, sidebar widget content |
| Label | 12px | 400 (regular) | 1.4 | Section headers ("Pomodoro", "Habits", "Journal"), streak counts, session counts, quote attribution, uppercase + 0.05em letter-spacing |
| Heading | 20px | 600 (semibold) | 1.2 | Not directly used in Phase 5 sidebar widgets (reserved for main content headings) |
| Display | 28px | 600 (semibold) | 1.2 | Pomodoro timer digits (minutes:seconds) displayed inside the circular ring |

Source: Phase 1 UI-SPEC typography scale. Display role repurposed for timer digits in this phase.

---

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#191919` | Page background, main content area, whiteboard container background |
| Secondary (30%) | `#252525` | Sidebar background (all widget sections inherit this), widget section backgrounds |
| Accent (10%) | `#5B9BD5` | See reserved-for list below |
| Destructive | `#E5484D` | Remove habit action, reset timer confirmation |

### Extended Palette (Phase 5 additions)

| Token | Value | Usage |
|-------|-------|-------|
| timer-ring-active | `#5B9BD5` | Active portion of pomodoro ring (work session) |
| timer-ring-break | `#30A46C` | Active portion of pomodoro ring (break session) |
| timer-ring-bg | `rgba(255, 255, 255, 0.08)` | Inactive/depleted portion of ring track |
| habit-complete | `#30A46C` | Filled habit dot for completed days |
| habit-incomplete | `rgba(255, 255, 255, 0.12)` | Empty habit dot for missed/future days |
| habit-streak | `#F5A623` | Streak count number when streak >= 7 days (highlight long streaks) |

### Accent Reserved For

Accent color `#5B9BD5` is used exclusively on these Phase 5 elements:
1. Pomodoro timer ring active arc (work session)
2. "Start today's journal" button text
3. Whiteboard sidebar button icon on hover
4. Active/selected journal entry in sidebar list
5. Pomodoro start button text when timer is paused/stopped

Accent is NOT used for: habit checkboxes (use text-primary), quote text, section headers, or ring background.

## Widget Layout Contracts

### Sidebar Widget Ordering (top to bottom within sidebar, below page tree)

1. Pomodoro Timer section
2. Habit Tracker section
3. Journal section
4. Whiteboard button (single row, not a section)
5. Quote footer (sticky to sidebar bottom)

Source: Claude's Discretion area from CONTEXT.md. Order follows urgency: timer is time-sensitive, habits are daily check-in, journal is reflective.

### Sidebar Widget Section Pattern

Every collapsible widget section follows this structure:

| Property | Value |
|----------|-------|
| Section header | 12px uppercase label, 0.05em letter-spacing, `text-secondary` color |
| Collapse trigger | Click anywhere on section header row |
| Collapse icon | `ChevronDown` (lucide) 16px, rotates to `ChevronRight` when collapsed |
| Collapse animation | 200ms ease-in-out on max-height |
| Section divider | 1px solid `rgba(255, 255, 255, 0.08)` above each section header |
| Section padding | 16px horizontal, 8px vertical within content area |
| Default state | All sections expanded on page load |

### Pomodoro Timer

```
+-----------------------------------+
|  POMODORO            [v]          |  <- section header, collapsible
|                                   |
|        +-----------+              |
|        |  ##:##    |              |  <- 96px ring, 28px digits center
|        +-----------+              |
|                                   |
|   [Start]  [Pause]  [Reset]      |  <- control buttons row
|                                   |
|   Sessions: 3          [gear]    |  <- session count + settings icon
+-----------------------------------+
```

| Property | Value |
|----------|-------|
| Ring diameter | 96px |
| Ring stroke width | 6px |
| Ring implementation | SVG `<circle>` with `stroke-dasharray` + `stroke-dashoffset` animation |
| Ring active color (work) | `#5B9BD5` |
| Ring active color (break) | `#30A46C` |
| Ring track color | `rgba(255, 255, 255, 0.08)` |
| Ring rotation | Starts at 12 o'clock (SVG transform rotate -90deg) |
| Ring animation | Smooth `stroke-dashoffset` transition, updates every second |
| Timer digits | 28px, weight 600, `text-primary`, centered inside ring |
| Control buttons | 32px height, 14px text, weight 400 |
| Start button | Text `#5B9BD5`, border 1px `rgba(91, 155, 213, 0.3)`, bg transparent |
| Pause button | Text `#F5A623`, border 1px `rgba(245, 166, 35, 0.3)`, bg transparent |
| Reset button | Text `text-secondary`, border 1px `rgba(255, 255, 255, 0.08)`, bg transparent |
| Button hover | Background `rgba(255, 255, 255, 0.05)`, 150ms transition |
| Button spacing | 8px gap between buttons, centered row |
| Session count | 12px label style, left-aligned |
| Settings icon | `Settings` (lucide) 16px, `text-secondary`, opens duration config |

### Duration Config (inline, expands below session count)

| Property | Value |
|----------|-------|
| Trigger | Click settings gear icon |
| Layout | Inline expansion below session count (not a modal) |
| Animation | 200ms ease-in-out slide down |
| Work duration | Number input, 12px label "Work (min)", default 25 |
| Break duration | Number input, 12px label "Break (min)", default 5 |
| Input style | 48px width, 28px height, `surface-elevated` bg, `border` border, `text-primary` text, 14px |
| Input range | 1-120 minutes |

### Habit Tracker

```
+-----------------------------------+
|  HABITS                [v]  [+]  |  <- section header + add button
|                                   |
|  [x] Meditate    5d  .......     |  <- checkbox, name, streak, 7 dots
|  [ ] Exercise    0d  .......     |
|  [ ] Read        12d .......     |
|                                   |
+-----------------------------------+
```

| Property | Value |
|----------|-------|
| Habit row height | 32px |
| Checkbox | 16px square, 2px border radius, `border` color border, `text-primary` check color |
| Checkbox checked | Fill `#30A46C` at 0.2 opacity background, `#30A46C` checkmark |
| Habit name | 14px, weight 400, `text-primary`, truncate with ellipsis at 100px max-width |
| Streak count | 12px, weight 400, `text-secondary` (or `#F5A623` when streak >= 7) |
| Streak suffix | "d" (e.g., "5d", "12d") |
| 7-day dots | 8px diameter circles, 4px gap |
| Dot complete | `#30A46C` fill |
| Dot incomplete | `rgba(255, 255, 255, 0.12)` fill |
| Dot today (incomplete) | `rgba(255, 255, 255, 0.08)` fill with 1px `border` stroke (distinguishes today from past) |
| Add habit button | `Plus` (lucide) 16px, `text-secondary`, hover `text-primary` |
| Add habit input | Inline text input that appears in-place when + clicked, 14px, `surface-elevated` bg |
| Habit context menu | Right-click or long-press: "Edit" / "Remove" options |
| Remove animation | 200ms ease-out slide left + fade |

### Daily Journal

```
+-----------------------------------+
|  JOURNAL               [v]       |  <- section header, collapsible
|                                   |
|  [Start today's journal]         |  <- CTA button (if no entry today)
|                                   |
|  March 19, 2026                  |  <- recent entry (clickable)
|  March 18, 2026                  |
|  March 17, 2026                  |
|  ... (last 7 entries)            |
+-----------------------------------+
```

| Property | Value |
|----------|-------|
| CTA button | Text "Start today's journal", 14px, weight 400, color `#5B9BD5` |
| CTA button style | Full-width, 36px height, bg transparent, border 1px `rgba(91, 155, 213, 0.2)`, border-radius 4px |
| CTA button hover | Background `rgba(91, 155, 213, 0.08)`, 150ms transition |
| CTA visibility | Hidden when today's journal entry already exists |
| Entry list | Last 7 entries, sorted newest first |
| Entry row | 14px, weight 400, `text-primary`, 32px height, 8px vertical padding |
| Entry row hover | Background `surface-elevated`, 150ms transition |
| Active entry | Text `#5B9BD5`, background `rgba(91, 155, 213, 0.08)` |
| Entry date format | "March 19, 2026" (full month name, day, year) |
| Click action | Opens journal entry in main content editor |

### Journal Entry Content (in editor, main content area)

| Property | Value |
|----------|-------|
| Title | Auto-set to date string (e.g., "March 19, 2026"), editable |
| Prompt 1 | "What are you grateful for?" -- rendered as H3 in editor |
| Prompt 2 | "What's your focus today?" -- rendered as H3 in editor |
| Prompt 3 | "How are you feeling?" -- rendered as H3 in editor |
| Prompt style | Each prompt as a heading followed by an empty paragraph for user input |

```
+-----------------------------------+

|  is to love what you do."         |
|  -- Steve Jobs                    |
|                                   |
+-----------------------------------+
```

| Property | Value |
|----------|-------|

### Whiteboard Button (sidebar)

```
+-----------------------------------+
|  [canvas icon]  Whiteboard   [>] |  <- single row, click to open
+-----------------------------------+
```

| Property | Value |
|----------|-------|
| Row height | 36px |
| Icon | `PenTool` (lucide) 16px, `text-secondary` |
| Label | "Whiteboard", 14px, weight 400, `text-primary` |
| Arrow | `ChevronRight` (lucide) 14px, `text-secondary` |
| Hover | Background `surface-elevated`, 150ms transition |
| Position | Between Journal section and Quote footer |
| Click action | Opens Excalidraw canvas full-width in main content area |

### Whiteboard (Main Content Area)

| Property | Value |
|----------|-------|
| Container | Full width and height of main content area (no max-width constraint) |
| Background | `#191919` |
| Loading | Skeleton shimmer (full container) while Excalidraw bundle loads lazily |
| Back button | Top-left, `ArrowLeft` (lucide) 20px + "Back to notes" text, 14px, `text-secondary` |

| Excalidraw toolbar | Default Excalidraw UI (all tools: selection, shapes, lines, freehand, text, arrows, eraser) |
| Lazy loading | Dynamic `import()` for `@excalidraw/excalidraw` -- bundle does not load until whiteboard is opened |
| Persist trigger | Save Excalidraw scene to Dexie settings table on scene change with 300ms debounce |

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | "Start today's journal" (journal section button) |
| Empty state: habits | "No habits yet. Tap + to add one." |
| Empty state: journal | "No journal entries yet. Start writing today." |
| Empty state: pomodoro sessions | "No sessions yet" (below timer, 12px text-secondary) |
| Error: timer notification | "Could not send notification. Check extension permissions." |
| Error: widget save failure | "Changes could not be saved. Check available disk space and try again." |
| Destructive: remove habit | "Remove habit": "Remove [habit name]? This deletes all tracking history." |
| Destructive: reset timer | No confirmation needed (reset is low-stakes, timer restarts) |
| Whiteboard loading | Skeleton shimmer, no text (Excalidraw renders its own UI once loaded) |
| Quote fallback | "The best time to start is now." -- Unknown (displayed if quote library fails to load) |

## Interaction States

### Widget Controls

| State | Style |
|-------|-------|
| Default | Per-element styles defined above |
| Focus-visible | 2px solid `#5B9BD5`, 2px offset, border-radius 4px |
| Hover | Per-element hover states defined above |
| Active/pressed | Opacity 0.8, 100ms transition |
| Disabled | Opacity 0.4, cursor not-allowed (e.g., Pause button when timer not running) |

### Keyboard Navigation

| Key | Action |
|-----|--------|
| `Space` / `Enter` | Toggle habit checkbox, activate buttons |
| `Tab` | Move focus through widget controls within sidebar |
| `Escape` | Close duration config if open, exit whiteboard back to notes |
| `Delete` / `Backspace` | Remove focused habit (with confirmation) |

## Animation Contract

All animations respect `prefers-reduced-motion: reduce` -- when enabled, all transitions resolve instantly (duration: 0ms).

| Animation | Duration | Easing | Property |
|-----------|----------|--------|----------|
| Widget section collapse | 200ms | ease-in-out | max-height, opacity |
| Pomodoro ring depletion | Per-second update | linear | stroke-dashoffset |
| Pomodoro ring session switch | 300ms | ease-in-out | stroke color |
| Habit checkbox toggle | 150ms | ease | background-color, border-color |
| Habit dot fill | 200ms | ease-out | fill opacity |
| Habit remove | 200ms | ease-out | transform (translateX), opacity |
| Journal CTA hover | 150ms | ease | background-color |
| Quote fade-in | 400ms | ease | opacity |
| Whiteboard skeleton shimmer | 1500ms | linear | background-position (infinite) |
| Duration config expand | 200ms | ease-in-out | max-height, opacity |
| Back button hover | 150ms | ease | color |
