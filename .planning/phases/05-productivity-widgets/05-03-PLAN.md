Create `components/widgets/HabitTracker.tsx`:

Import: `useHabitStore`, `CollapsibleSection`, `Plus` from lucide-react, `useState`, `useEffect`, `useRef`.

State:
- `const [isAdding, setIsAdding] = useState(false)` -- inline add mode
- `const [newName, setNewName] = useState('')` -- new habit name input
- `const [editingId, setEditingId] = useState<string | null>(null)` -- editing habit name
- `const [editName, setEditName] = useState('')`
- `const [contextMenuId, setContextMenuId] = useState<string | null>(null)` -- right-click menu
- `const [contextMenuPos, setContextMenuPos] = useState({ x: 0, y: 0 })`
- `const inputRef = useRef<HTMLInputElement>(null)`

On mount: Call `useHabitStore.getState().loadFromStorage()`.

**Structure wrapped in CollapsibleSection:**

```tsx
<CollapsibleSection
  id="habits"
  title="HABITS"
  rightAction={
    <button
      onClick={(e) => { e.stopPropagation(); setIsAdding(true); }}
      className="text-text-secondary hover:text-text-primary transition-colors duration-150 focus-visible:ring-2 focus-visible:ring-accent focus-visible:ring-offset-2 focus-visible:ring-offset-bg rounded-[4px]"
      aria-label="Add habit"
    >
      <Plus size={16} />
    </button>
  }
>
```

**Empty state** (when habits.length === 0 and !isAdding):
- `<p className="text-[14px] text-text-secondary">No habits yet. Tap + to add one.</p>`

**Add habit input** (when isAdding):
- `<input ref={inputRef} value={newName} onChange={...} onKeyDown={handleAddKeyDown} className="w-full h-[32px] bg-surface-elevated border border-border text-text-primary text-[14px] px-[8px] rounded-[4px] outline-none focus:border-accent" placeholder="Habit name..." autoFocus />`
- On Enter: if newName.trim(), call addHabit(newName.trim()), setNewName(''), setIsAdding(false)
- On Escape: setNewName(''), setIsAdding(false)
- Focus input on mount via useEffect when isAdding changes

**Habit list** (map over habits):

Each habit row: `<div key={habit.id} className="flex items-center h-[32px] gap-[8px]" onContextMenu={handleContextMenu(habit.id)}>`:

1. **Checkbox** (16px square):
```tsx
<button
  onClick={() => toggleToday(habit.id)}
  className={`w-[16px] h-[16px] shrink-0 rounded-[2px] border transition-colors duration-150 flex items-center justify-center focus-visible:ring-2 focus-visible:ring-accent focus-visible:ring-offset-2 focus-visible:ring-offset-bg ${
    isCheckedToday
      ? 'bg-habit-complete/20 border-habit-complete'
      : 'border-border-strong bg-transparent'
  }`}
  aria-label={`Toggle ${habit.name}`}
>
  {isCheckedToday && (
    <svg width={10} height={8} viewBox="0 0 10 8" fill="none">
      <path d="M1 4L3.5 6.5L9 1" stroke="#30A46C" strokeWidth={1.5} strokeLinecap="round" strokeLinejoin="round" />
    </svg>
  )}
</button>
```

Where `isCheckedToday = habit.completions[new Date().toISOString().slice(0, 10)] === true`.

2. **Habit name** (if editing this habit, show input; else show text):
   - Text: `<span className="text-[14px] text-text-primary truncate max-w-[100px]">{habit.name}</span>`
   - Edit input: `<input value={editName} onChange={...} onKeyDown={handleEditKeyDown} onBlur={commitEdit} className="flex-1 h-[24px] bg-surface-elevated border border-border text-text-primary text-[14px] px-[4px] rounded-[4px] outline-none focus:border-accent" autoFocus />`
   - On Enter in edit: call renameHabit(editingId, editName.trim()), clear editingId
   - On Escape: clear editingId

3. **Streak count**:
```tsx
<span className={`text-[12px] shrink-0 ${streak >= 7 ? 'text-habit-streak' : 'text-text-secondary'}`}>
  {streak}d
</span>
```
Where `streak = getStreak(habit)`.

4. **7-day dots** (row of 7 circles):
```tsx
<div className="flex gap-[4px] shrink-0">
  {getLast7Days(habit).map((completed, i) => {
    const isToday = i === 6;
    return (
      <div
        key={i}
        className={`w-[8px] h-[8px] rounded-full transition-[fill-opacity] duration-200 ease-out ${
          completed
            ? 'bg-habit-complete'
            : isToday
            ? 'bg-timer-ring-bg border border-border-strong'
            : 'bg-habit-incomplete'
        }`}
      />
    );
  })}
</div>
```

**Context menu** (absolute positioned, shown when contextMenuId is set):
```tsx
{contextMenuId && (
  <div
    className="fixed bg-surface-elevated border border-border rounded-[4px] shadow-lg py-[4px] z-50"
    style={{ left: contextMenuPos.x, top: contextMenuPos.y }}
  >
    <button
      onClick={() => { setEditingId(contextMenuId); setEditName(habits.find(h => h.id === contextMenuId)?.name ?? ''); setContextMenuId(null); }}
      className="w-full text-left px-[12px] py-[6px] text-[14px] text-text-primary hover:bg-white/5 transition-colors duration-150"
    >
      Edit
    </button>
    <button
      onClick={() => { if (confirm(`Remove ${habits.find(h => h.id === contextMenuId)?.name}? This deletes all tracking history.`)) { removeHabit(contextMenuId); } setContextMenuId(null); }}
      className="w-full text-left px-[12px] py-[6px] text-[14px] text-destructive hover:bg-white/5 transition-colors duration-150"
    >
      Remove
    </button>
  </div>
)}
```

- `handleContextMenu(id)` = `(e: React.MouseEvent) => { e.preventDefault(); setContextMenuId(id); setContextMenuPos({ x: e.clientX, y: e.clientY }); }`
- Close context menu on click outside: useEffect with document click listener when contextMenuId is set.

**Remove animation:** Wrap each habit row in a div that gets `transition-all duration-200 ease-out`. On remove, the list re-renders without the item (React handles this naturally).

**Keyboard support:**
- Delete/Backspace on focused habit row: trigger remove with confirmation
- Tab navigates between checkboxes
- All interactive elements are `<button>` for native keyboard support
