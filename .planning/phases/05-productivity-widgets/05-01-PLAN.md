**2. Create `lib/stores/pomodoro-store.ts`:**

Zustand store following the existing pattern (create<State>((set, get) => ...)). State shape:

```typescript
interface PomodoroStoreState {
  workDuration: number;        // default 25
  breakDuration: number;       // default 5
  sessionsCompleted: number;   // default 0
  isRunning: boolean;          // default false
  isBreak: boolean;            // default false
  timeRemaining: number;       // seconds, default 25*60
  intervalId: ReturnType<typeof setInterval> | null;

  // Actions
  start: () => void;
  pause: () => void;
  reset: () => void;
  tick: () => void;
  setWorkDuration: (mins: number) => void;
  setBreakDuration: (mins: number) => void;
  onSessionComplete: () => void;
  loadFromStorage: () => Promise<void>;
  persist: () => Promise<void>;
}
```

- `start()`: Sets isRunning=true, creates setInterval calling tick() every 1000ms, stores intervalId
- `pause()`: Sets isRunning=false, clears interval
- `reset()`: Clears interval, sets isRunning=false, timeRemaining = workDuration*60, isBreak=false
- `tick()`: Decrements timeRemaining by 1. If timeRemaining reaches 0, calls onSessionComplete()
- `onSessionComplete()`: If !isBreak, increments sessionsCompleted, switches to break (isBreak=true, timeRemaining=breakDuration*60). If isBreak, switches to work (isBreak=false, timeRemaining=workDuration*60). Calls persist(). Does NOT auto-start -- sets isRunning=false and clears interval.
- `setWorkDuration(mins)`: Clamps 1-120. Updates workDuration. If not running and not break, also set timeRemaining = mins*60. Calls persist().
- `setBreakDuration(mins)`: Clamps 1-120. Updates breakDuration. If not running and isBreak, also set timeRemaining = mins*60. Calls persist().
- `loadFromStorage()`: Reads `db.settings.get('pomodoro-state')`. If found, restores workDuration, breakDuration, sessionsCompleted. Does NOT restore isRunning or timeRemaining (timer resets on page load).
- `persist()`: Writes `{ key: 'pomodoro-state', value: { workDuration, breakDuration, sessionsCompleted, updatedAt: Date.now() } }` to `db.settings.put(...)`.

Import db from `../storage/db`.

**3. Create `lib/stores/habit-store.ts`:**

```typescript
interface HabitStoreState {
  habits: HabitRecord[];
  initialized: boolean;

  loadFromStorage: () => Promise<void>;
  persist: () => Promise<void>;
  addHabit: (name: string) => void;
  removeHabit: (id: string) => void;
  renameHabit: (id: string, name: string) => void;
  toggleToday: (id: string) => void;
  getStreak: (habit: HabitRecord) => number;
  getLast7Days: (habit: HabitRecord) => boolean[];
}
```

- `loadFromStorage()`: Reads `db.settings.get('habit-data')`. Parses value as HabitData. Sets habits array.
- `persist()`: Writes `{ key: 'habit-data', value: { habits: get().habits, updatedAt: Date.now() } }` to db.settings.put.
- `addHabit(name)`: Creates HabitRecord with `id: crypto.randomUUID(), name, createdAt: Date.now(), completions: {}`. Appends to habits array. Calls persist().
- `removeHabit(id)`: Filters out habit with matching id. Calls persist().
- `renameHabit(id, name)`: Maps over habits, updates name for matching id. Calls persist().
- `toggleToday(id)`: Gets today's date key as `new Date().toISOString().slice(0, 10)` (YYYY-MM-DD format). Toggles `completions[dateKey]`. Calls persist().
- `getStreak(habit)`: Counts consecutive days backward from yesterday (NOT today, since today may be incomplete). Loop: start from yesterday, check if `completions[dateKey]` is true, increment streak, move to previous day. Stop at first miss. Return count.
- `getLast7Days(habit)`: Returns array of 7 booleans, index 0 = 6 days ago, index 6 = today. For each day, check `completions[dateKey]`.

Helper for date key: `const toDateKey = (d: Date) => d.toISOString().slice(0, 10);`

Import db from `../storage/db`, HabitRecord from `../storage/types`.
