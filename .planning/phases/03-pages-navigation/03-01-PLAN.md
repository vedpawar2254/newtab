**1. Create `lib/utils/tree-utils.ts`** with these pure functions:

```typescript
import type { TreeIndexEntry } from '../storage/types';

export interface FlattenedItem {
  id: string;
  parentId: string | null;
  depth: number;
  index: number;
  title: string;
  childCount: number;
  order: number;
}
```

- `flattenTree(entries: TreeIndexEntry[], parentId: string | null = null, depth: number = 0): FlattenedItem[]` — Recursively flattens tree. Filters entries by parentId, sorts by `order`, returns flat array where each item has its `depth`. `childCount` = total descendant count (not just direct children). `index` = position within flat array.

- `getDescendantCount(entries: TreeIndexEntry[], parentId: string): number` — Returns total number of descendants (all levels) for a given parent.

- `removeChildrenOf(items: FlattenedItem[], ids: string[]): FlattenedItem[]` — Removes all descendants of the given ids from the flattened list. Used to hide children of collapsed items. For each id in ids, find its index, then skip forward by its childCount items.

- `collectSubtreeIds(entries: TreeIndexEntry[], rootId: string): string[]` — Returns all descendant IDs (not including rootId). Recursive: get children of rootId, then children of those children, etc.

- `reindexSiblings(entries: TreeIndexEntry[], parentId: string | null): TreeIndexEntry[]` — Returns a new entries array where all entries with the given parentId have their `order` field reset to 0, 1, 2... based on their current relative order. Other entries unchanged.

```tsx
import { Plus } from 'lucide-react';
import { useNotesStore } from '../../lib/stores/notes-store';
import { useUIStore } from '../../lib/stores/ui-store';

export function NewPageButton() {
  const createNote = useNotesStore(s => s.createNote);
  const setActiveNote = useUIStore(s => s.setActiveNote);
  const setRenamingId = useUIStore(s => s.setRenamingId);

  const handleClick = async () => {
    const note = await createNote('Untitled', null);
    setActiveNote(note.id);
    setRenamingId(note.id); // auto-focus rename on new page
  };

  return (
    <button
      onClick={handleClick}
      aria-label="Create new page"
      className="flex items-center gap-[8px] h-[32px] px-[8px] w-full rounded text-accent text-[12px] font-normal uppercase tracking-[0.05em] hover:bg-surface-elevated transition-colors duration-150"
    >
      <Plus size={14} />
      <span>New Page</span>
    </button>
  );
}
```
