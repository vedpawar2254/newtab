# Phase 3 — UI Design Contract

> Visual and interaction contract for the Pages + Navigation phase. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none |
| Preset | not applicable |
| Component library | radix-ui (primitives: dialog for delete confirmation) |
| Icon library | lucide-react |

## Spacing Scale

Inherited from Phase 1. Declared values (multiples of 4):

| Token | Value | Usage in Phase 3 |
|-------|-------|-------------------|
| xs | 4px | Gap between chevron and page name, icon gaps |
| sm | 8px | Vertical padding on tree items, gap between drag handle and chevron |
| md | 16px | Nesting indentation per level, horizontal padding on tree items |
| lg | 24px | Sidebar section padding (above/below tree area) |
| xl | 32px | Not used in Phase 3 |
| 2xl | 48px | Not used in Phase 3 |
| 3xl | 64px | Not used in Phase 3 |

Exceptions:
- Nesting indentation: 16px per level (1st level = 16px left padding, 2nd = 32px, 3rd = 48px, etc.)
- Drag handle zone: 20px width to the left of the page name (houses the grip icon on hover)
- Chevron hit target: 20px x 20px (accessible touch/click area, icon renders at 14px)
- Drop indicator line: 2px height, full width of the tree area minus current indent

Accent color `#5B9BD5` is used exclusively on these elements:
1. Active page row left border indicator (2px)
2. Drop position indicator line during drag-and-drop
3. "New Page" button text and icon
4. Hover "+" icon on page rows (create child page)
5. Focus ring on inline rename input (2px solid)

Accent is NOT used for: chevron icons, drag handles, context menu backgrounds, delete confirmation backgrounds, or any large surface.

## Layout Contract

### Sidebar Tree Structure

```
+------------------------------------------+
| [+ New Page]                    (always)  |
|------------------------------------------|
| > Page A                        [. . .]  |
|   > Subpage A1                  [. . .]  |
|     Subpage A1a                 [. . .]  |
|   Subpage A2                    [. . .]  |
| Page B  (active)  |||           [. . .]  |
| > Page C                        [. . .]  |
|                                           |
| (remaining space)                         |
+------------------------------------------+

Legend:
  >       = Collapsed chevron (has children)
  v       = Expanded chevron (has children, showing)
  (none)  = No chevron (leaf page)
  [. . .] = "More" menu icon (visible on hover only)
  |||     = Left accent border (active page)
```

### Tree Item Row

| Property | Value |
|----------|-------|
| Height | 32px (fixed, single-line) |
| Horizontal padding | 8px left (before indent), 8px right |
| Vertical padding | Centered within 32px height |
| Indent per level | 16px per nesting level |
| Background (default) | Transparent |
| Background (hover) | `#2F2F2F` |
| Background (active) | `rgba(91, 155, 213, 0.08)` |
| Active indicator | 2px left border, `#5B9BD5` |
| Text color | `#ECECEC` |
| Text overflow | Ellipsis, single line |
| Transition | 150ms background-color ease |

### Tree Item Anatomy (left to right)

```
[indent] [drag-handle] [chevron] [page-name] ... [hover-actions]
```

| Element | Size | Visibility |
|---------|------|------------|
| Indent spacer | 16px x nesting level | Always |
| Drag handle | `GripVertical` 14px, color `rgba(255,255,255,0.30)` | Visible on row hover only |
| Chevron | `ChevronRight` 14px, color `rgba(255,255,255,0.40)` | Visible only if page has children |
| Page name | 14px body text | Always |
| "+" create child | `Plus` 14px, color `#5B9BD5` | Visible on row hover only |
| "..." more menu | `MoreHorizontal` 14px, color `rgba(255,255,255,0.40)` | Visible on row hover only |

### "New Page" Button

| Property | Value |
|----------|-------|

| Height | 32px |
| Padding | 8px horizontal |

## Drag-and-Drop Contract

### Drag Initiation

| Property | Value |
|----------|-------|
| Trigger | Mousedown on drag handle icon (`GripVertical`) |
| Drag delay | 0ms (immediate on handle grab) |
| Cursor | `grabbing` during drag |
| Minimum drag distance | 4px before drag activates (prevents accidental drags) |

### Ghost Element

| Property | Value |
|----------|-------|
| Content | Page name text only (no chevron, no action icons) |
| Background | `rgba(37, 37, 37, 0.90)` |
| Border | 1px solid `rgba(255, 255, 255, 0.12)` |
| Border-radius | 4px |
| Padding | 4px 12px |
| Font | 14px, weight 400, color `#ECECEC` |
| Opacity | 1.0 |
| Shadow | none (flat design) |

### Drop Indicators

| Indicator | Style | When Shown |
|-----------|-------|------------|
| Between pages (reorder) | 2px horizontal line, `#5B9BD5`, full tree width minus indent | Cursor over top/bottom 25% of a row |
| Nest as child (indent) | Entire target row background `rgba(91, 155, 213, 0.06)` + 2px left border `#5B9BD5` | Cursor over right 50% of a row |
| Invalid drop | No indicator shown | Dragging page onto itself or its own descendant |

### Collapsed Parent Auto-Expand

| Property | Value |
|----------|-------|
| Hover delay | 500ms |
| Expand animation | Chevron rotates 90deg over 150ms ease, children slide in over 200ms ease-out |
| Behavior | Collapsed parent auto-expands after hovering 500ms during an active drag |

### Subtree Behavior

- Dragging a parent moves its entire subtree together
- Drop indicators account for subtree size (cannot drop between a parent and its first child unless nesting)
- Visual feedback: dragged subtree children fade to opacity 0.3 in the tree during drag

| Button | Style |

## Interaction States

### Tree Item States

| State | Visual |
|-------|--------|
| Default | Transparent background, `#ECECEC` text |
| Hover | `#2F2F2F` background, drag handle + "+" + "..." icons appear |
| Active (selected page) | `rgba(91, 155, 213, 0.08)` background, 2px left border `#5B9BD5` |
| Active + Hover | `rgba(91, 155, 213, 0.12)` background, 2px left border `#5B9BD5`, hover icons appear |
| Dragging (source) | Opacity 0.3 |
| Drop target (reorder) | 2px blue line above or below |
| Drop target (nest) | `rgba(91, 155, 213, 0.06)` background + 2px left border |
| Renaming | Inline input replaces name text, focus ring visible |

### Keyboard Navigation

| Key | Context | Action |
|-----|---------|--------|
| `ArrowDown` | Sidebar tree focused | Move focus to next visible tree item |
| `ArrowUp` | Sidebar tree focused | Move focus to previous visible tree item |
| `ArrowRight` | Collapsed page with children | Expand children |
| `ArrowRight` | Expanded page with children | Move focus to first child |
| `ArrowLeft` | Expanded page with children | Collapse children |
| `ArrowLeft` | Child page or leaf | Move focus to parent |
| `Enter` | Tree item focused | Navigate to that page (load in editor) |
| `F2` | Tree item focused | Begin inline rename |
| `Delete` / `Backspace` | Tree item focused | Delete page (with confirmation if has children) |
| `Escape` | Inline rename active | Cancel rename |
| `Escape` | Context menu open | Close menu |
| `Escape` | Delete dialog open | Dismiss dialog (cancel) |

## Animation Contract

All animations respect `prefers-reduced-motion: reduce` -- when enabled, all transitions resolve instantly (duration: 0ms).

| Animation | Duration | Easing | Property |
|-----------|----------|--------|----------|
| Tree item hover | 150ms | ease | background-color |
| Chevron rotate | 150ms | ease | transform (rotate) |
| Children reveal | 200ms | ease-out | height, opacity |
| Children hide | 150ms | ease-in | height, opacity |
| Children stagger (reveal) | 30ms per child | ease-out | opacity, transform |
| Drag ghost follow cursor | 0ms | none | transform (follows pointer) |
| Drop indicator appear | 100ms | ease-out | opacity |
| Auto-expand on drag hover | 500ms delay, then expand animation | -- | -- |
| Context menu enter | 100ms | ease-out | opacity, transform (scale) |
| Context menu exit | 75ms | ease-in | opacity |
| Dialog overlay enter | 200ms | ease-out | opacity |
| Dialog enter | 150ms | ease-out | opacity, transform (scale) |
| Dialog exit | 100ms | ease-in | opacity |
| Hover icon appear ("+", "...", grip) | 100ms | ease | opacity |
| Dragged subtree children fade | 150ms | ease | opacity |
| Page navigation after delete | Instant (no animation, immediately select next page) |

## Post-Delete Navigation

When the currently active page is deleted, navigate to the next logical page:

| Scenario | Navigate To |
|----------|-------------|
| Deleted page has a sibling below | Next sibling |
| Deleted page has a sibling above only | Previous sibling |
| Deleted page has no siblings (only child) | Parent page |
| Deleted page was the last page in the tree | Show empty state |

Navigation is instant with no transition animation.
